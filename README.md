# Sysnordic – statisk nettside (Bootstrap)

Dette er en ren statisk nettside (HTML/CSS/JS) uten backend.

## Kjør lokalt
- Åpne `index.html` i nettleseren, eller
- Kjør en enkel lokal webserver (anbefalt):
  - Python: `python -m http.server 8000`
  - Gå til: http://localhost:8000

## Struktur
- `index.html` – Hjem
- `tjenester.html` – Tjenester
- `compliance.html` – Rammeverk/etterlevelse
- `om-oss.html` – Om oss
- `kontakt.html` – Kontakt (statisk skjema)

## Teknologi
- Bootstrap 5 (CDN)
- Google Fonts: Space Grotesk + Share Tech Mono
- Egendefinert glass/CRT-stil i `assets/css/styles.css`
- Mikro-animasjoner i `assets/js/main.js`
